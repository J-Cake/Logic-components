# NOR-Gate

* **Inputs - 2:** `0` `1`
* **Outputs - 1:** `o`
* **Type:** Stateless
* **Token:** `$nor`

## Synopsis: 
The *Nor* gate produces an output only when **either** of its inputs are active.
