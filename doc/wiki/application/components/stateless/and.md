# AND-Gate

* **Inputs - 2:** `0` `1`
* **Outputs - 1:** `o`
* **Type:** Stateless
* **Token:** `$and`

## Synopsis: 
The *And* gate produces an output only when **both** of its inputs are active.

## Truth Table

|---------|---------|--------|
| Off     | Off     | Off    |
| Active  | Off     | Off    |
| Off     | Active  | Off    |
| Active  | Active  | Active |
