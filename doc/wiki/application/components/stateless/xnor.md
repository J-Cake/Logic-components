# XNOR-Gate

* **Inputs - 2:** `0` `1`
* **Outputs - 1:** `o` 
* **Type:** Stateless
* **Token:** `$xnor`

## Synopsis: 
The *Xnor* gate produces an output only when **both** of its inputs are **equal**.
